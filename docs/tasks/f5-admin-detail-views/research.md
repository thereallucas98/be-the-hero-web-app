# Research — Admin Detail Views

## Decision 1 — Pets, workspaces, campaigns: separate detail pages
Click row → navigate to detail page. Approve/reject buttons on detail page.

## Decision 2 — Donations, follow-ups: expandable rows
Click to expand inline — shows extra fields. Keeps the queue flow fast without page navigation.

## Decision 3 — Back link on detail pages
Standard "Voltar" link back to the queue list.
